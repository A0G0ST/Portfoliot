Bildgenereringsapp

Image generator app

Byggd under utbildningen

Detta projektet är en enkel men komplett crossplattform-applikation byggd i Flutter. Applikationen fungerar på både Android och webben. Appen består av två huvudsidor: en login-sida och en hemsida. På login-sidan kan användaren fylla i e-post och lösenord (fiktiv inloggning). När man loggat in navigeras man vidare till hemsidan, där appen hämtar och visar bilder på hundar från The Dog API via HTTP-anrop.

Built during my studies

This project is a simple yet complete cross-platform application built with Flutter. The application runs on both Android and the web. The app consists of two main pages: a login page and a home page. On the login page, the user can enter their email and password (fictional login). Once logged in, the user is navigated to the home page, where the app fetches and displays pictures of dogs from The Dog API via HTTP requests.
